# Kuli Tinta AI

News Generator PWA untuk jurnalis Indonesia, powered by Gemini 3 Pro.

## Prerequisites

- Node.js 18+
- npm atau yarn

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Build (Static Export)

```bash
npm run build
```

Hasil build berada di folder `out/`. Deploy ke hosting statis (Vercel, Netlify, dll).

## Features

- ✅ BYOK (Bring Your Own Key) - API Key tersimpan lokal
- ✅ Draft History - Tersimpan di IndexedDB
- ✅ Newspaper-style Preview
- ✅ Static Export (No Backend)
